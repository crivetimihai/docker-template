# docker-template
Docker Template and Build Script (Node.JS sample)

# Summary
- Alpine Linux - Edge with Node.JS and NPM (template)
- `build.sh` docker build script

# Usage:
```
sudo mkdir -p /volumes/node
./build.sh build
./build.sh shell
logout
./build.sh stop
./build.sh rm
./build.sh clean
```
